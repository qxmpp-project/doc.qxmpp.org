Detailed Description

The QXmppOmemoDeviceBundle class represents an OMEMO bundle as defined by XEP-0384: OMEMO Encryption.

It is a collection of publicly accessible data used by the X3DH key exchange. The data is used to build an encrypted session with an OMEMO device.

QXmpp 1.5
