Detailed Description

QXmppDiscoveryIq represents a discovery IQ request or result containing a list of features and other information about an entity as defined by XEP-0030: Service Discovery.

Member Enumeration Documentation

◆ QueryType

enum QXmppDiscoveryIq::QueryType

Specifies the type of a service discovery query. An InfoQuery queries identities and features, an ItemsQuery queries subservices in the form of items.
